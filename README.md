## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/felipeantoniob/TS-808.git
   ```
2. Install packages

   ```sh
   npm install
   ```

### Usage

1. Run the development server

   ```sh
   npm run dev
   ```

2. Open http://localhost:5173 with your browser to see the result.
